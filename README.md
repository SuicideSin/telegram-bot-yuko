# Telegram Bot Yuko

[![Greenkeeper badge](https://badges.greenkeeper.io/preco21/telegram-bot-yuko.svg)](https://greenkeeper.io/)

> 플비의 커여운 딸내미

## 설치하기

```bash
$ npm run install
```

## 개발하기

```bash
$ npm run dev
```

## 빌드하기

```bash
$ npm run build
```

## 실행하기

> 실행하려면 빌드를 먼저 해야 합니다.

```bash
$ npm start
```

## 배치하기

```bash
$ npm run deploy
```
